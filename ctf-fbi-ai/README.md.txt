A small FBI-themed AI chatbot made for a prompt-injection CTF challenge.

## How to run

1. Install Python 3.10+
2. Install Flask:

3. Run the server:
   python app.py

4. Open in browser:
   http://127.0.0.1:8000

## Goal

Interact with the AI and use prompt injection to force it to reveal the flag.

The flag is revealed only through the chat interface.
